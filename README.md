Асинхронный парсер для Яндекс.Карт — собирает подробную информацию об организациях в выбранной области поиска.

## Используемый стек
- Python 3.13
- Playwright + Asyncio
- Json

## Установка и запуск
~~~
git clone https://github.com/tak3e/yandexmapsgraber.git
cd YandexMapsGraber
pip3 install -r requirements.txt
python3 main.py
~~~
